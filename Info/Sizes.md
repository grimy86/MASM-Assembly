|------------|-------------------|----------|
| BYTE       | 1 byte            | 8 bits   |
| WORD       | 2 bytes           | 16 bits  |
| DWORD      | 4 bytes           | 32 bits  |
| QWORD      | 8 bytes           | 64 bits  |

| Megabyte   | 1,048,576 bytes   |          |
